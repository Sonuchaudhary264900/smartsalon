# Smart Salon Backend

A Node.js + Express + MongoDB backend for salon booking system.

## Features
- JWT Authentication
- Role-based access control
- Salon creation & approval
- Service management
- Booking system
- Cancel booking
- Earnings calculation

## Tech Stack
- Node.js
- Express
- MongoDB
- Mongoose
- JWT

## Setup Instructions

1. Clone the repository
2. Run `npm install`
3. Create `.env` file
4. Add:

MONGO_URL=your_mongodb_connection_string

5. Run `node server.js`

Server runs on port 5000
